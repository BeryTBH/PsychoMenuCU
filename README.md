# PsychoMenuCU
PsychoMenuCU is a [BepInEx](https://github.com/BepInEx/BepInEx) Classic Us mod built with the intention for the (most likely public) first ever cheat menu.

# Installation and Usage

## Installing BepInEx
[BepInEx](https://github.com/BepInEx/BepInEx) is a modding framework that allows you to create mods for Unity games. Classic Us is based on the old among us unity project. PsychoMenuCU requires [BepInEx](https://github.com/BepInEx/BepInEx) to work, to install it, you will need to download it from here: https://builds.bepinex.dev/projects/bepinex_be/785/BepInEx-Unity.IL2CPP-win-x64-6.0.0-be.785%2B6abdba4.zip then extract the downloaded ZIP and then you will need to put the files inside the extracted folder into the Classic Us folder.

## Installing PsychoMenuCU
Go to the [Releases](https://github.com/BeryTBH/PsychoMenuCU/releases) tab and download the latest `PsychoMenuCU.dll` file, and put the file into the `./BepInEx/plugins/` directory. After you have done that launch your game, for the first launch with BepInEx it should take a while. Once loaded in, if it worked Classic Us should have opened.

## Using PsychoMenuCU
Open the menu (not in the loading screen, if you do the buttons will have a transparent background) by pressing `INSERT` on your keyboard. Congrats! you can now use the menu.
